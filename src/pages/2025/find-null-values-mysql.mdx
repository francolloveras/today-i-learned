You can use `IS NULL` in a `SELECT` query to find null values.

```sql
SELECT * FROM your_table WHERE your_column IS NULL;
```

To search non-null values use `IS NOT NULL`.

```sql
SELECT * FROM your_table WHERE your_column IS NOT NULL;
```
