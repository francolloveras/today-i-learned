You can use [BrowserFrame](https://browserframe.com). First insert the resource, you can use a website URL or upload an image file. then choose a browser frame, you can choose an existing web browser or use a generic one.

Also is possible to specify some properties like padding, background color, shadows, heigth, width, etc.
